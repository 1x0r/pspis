Машинное обучение

(Machine Learning)

Машинное обучение
направление в науке об искусственном интеллекте, изучающее способность компьютеров к обучению

Машинное обучение по Т. Митчеллу
Говорится, что алгоритм $A$ обучается на опыте $\mathbf{D}$ для решения класса задач $T$, если его производительность, определяемая по функционалу качества $\mathbf{L}$, увеличивается с опытом $\mathbf{D}$

Терминология

  • $X$ — пространство входов
  • $Y$ — пространство выходов
  • $\mathbf{D}$ — множество данных (dataset)
  • $f: X \to Y$ — гипотеза, функция предсказания
  • $F$ — множество всех гипотез $f$ для $D$
  • $A: X^n \to F$ — алгоритм обучения, т.е. выбора $f \in F$ на основе $D$
  • $\mathbf{D}$ — множество данных (dataset)
  • $\mathbf{L}$ — функция стоимости (loss function)

Парадигмы машинного обучения

Примеры задач машинного обучения

  • Распознавание рукописного текста
  • Обнаружение спама
  • Машинный перевод
  • Распознавание изображений
  • Управление автономными роботами
  • Рекомендательные системы
  • Игры

Парадигмы обучения

  • Обучение с учителем (supervised)
  • Обучение с привлечением учителя (semi-supervised)
  • Обучение без учителя (unsupervised)
  • Обучение с подкреплением (reinforcement)
Обучение с учителем
класс алгоритмов, обучающихся на маркированных примерах: множество $\mathbf{D}$ содержит для каждого входа $x_i \in X$ соответствующий ему выход $y_i \in Y$

Решаемые задачи:

  • Регрессия
  • Классификация
Обучение с привлечением учителя
класс алгоритмов, обучающихся как на маркированных, так и на немаркированных примерах: множество $\mathbf{D}$ не для каждого входа $x_i \in X$ содержит соответствующий ему выход $y_i \in Y$

Решаемые задачи:

  • Ранжирование
  • Трансдукция
Обучение без учителя
класс алгоритмов, обучающихся на немаркированных примерах: множество $\mathbf{D}$ содержит только примеры $x_i \in X$

Решаемые задачи:

  • Кластеризация
  • Уменьшение размерности
  • Поиск аномалий
Обучение с подкреплением
класс алгоритмов, обучающихся на реакциях среды: множество $\mathbf{D}$ получается в процессе взаимодействия алгоритма (агента) со средой

Решаемые задачи:

  • Управление
  • Логический вывод

Задача регрессии
